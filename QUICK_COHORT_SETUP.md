# Quick Cohort Setup for Instructors

## 🚀 Create Your First Cohort in 3 Steps

### Step 1: Login & Navigate
```
1. Login as instructor
2. Go to /admin/cohorts
3. Click "Create Cohort" button
```

### Step 2: Fill Required Fields
```
✅ Cohort Name: "Web Dev Bootcamp Q1 2024"
✅ Start Date: Select from calendar
✅ End Date: Select from calendar
```

### Step 3: Submit
```
Click "Create Cohort" → Done! ✨
```

---

## 📋 Quick Form Guide

### Required Fields (*)
- Cohort Name
- Start Date
- End Date

### Optional Settings
- Description
- Status (default: upcoming)
- Max Learners (default: 30)
- Performance Threshold (default: 70%)
- Weekly Target (default: 10 hrs)
- Grace Period (default: 3 days)
- Review Cycle (default: weekly)

## ⚡ Quick Actions

### Edit Cohort
`Click "Edit" button → Update → Save`

### Delete Cohort
`Click trash icon → Confirm → Deleted`

### Change Status
`Edit → Select new status → Save`

---

## 🔧 Troubleshooting

### Can't see "Create Cohort" button?
→ Check your role is instructor/admin

### "Failed to create cohort"?
→ Ensure cohort name is unique

### Cohort not appearing?
→ Refresh the page
